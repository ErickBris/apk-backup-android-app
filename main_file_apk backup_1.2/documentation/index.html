categorize in package

All the java code is categorize in several package.
We have one activity placed on com.app.apkbackup --> 

ActivityMain.java --> this activity handle first launch application

1. com.app.apkbackup.adapter 

adapter handle all custom listview item. and fragment adapter for tabber.

 

2. com.app.apkbackup.data

in package data handle constanta and any Utilities operations.

 

3. com.app.apkbackup.fragment

thi package consist two clas backup fragment and restore fragment, this fragment view in two page.

 

4. com.app.apkbackup.model

in model we have two model object for support listview.

Change Color

For all Color Data you can edit in resources/ value/ colors.xml

<?xml version="1.0" encoding="utf-8"?>
<resources>

    <color name="colorPrimary">#1565C0</color>
    <color name="colorPrimaryDark">#0D47A1</color>
    <color name="colorPrimaryLight">#90CAF9</color>

    <color name="colorAccent">#FF5722</color>
    <color name="colorAccentDark">#E64A19</color>
    <color name="colorAccentLight">#80D8FF</color>
    <color name="textColorSecondary">@color/grey_hard</color>
